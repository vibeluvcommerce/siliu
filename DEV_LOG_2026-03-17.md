## 🎯 分步开发计划

### Step 1: 验证 JS 注入基础能力
**目标**: 确认 BrowserView 支持 JS 注入蒙版

**任务**:
1. 创建测试按钮（临时）
2. 实现最简蒙版注入 IPC
3. 在 test.com 等简单网站测试
4. 记录 CSP 限制表现

**代码范围**:
- `src/app.js` - 添加 `annotation:inject` handler
- `src/preload/index.js` - 暴露 `injectAnnotation()` API
- `public/shell.html` - 临时测试按钮

**成功标准**:
- 点击按钮后页面显示红色半透明蒙版
- 点击蒙版弹出 alert 显示坐标
- 确认 z-index 足够高

---

### Step 2: 坐标捕获与存储
**目标**: 点击蒙版记录坐标到内存

**任务**:
1. 蒙版点击发送 postMessage 到 shell
2. shell 接收并弹出命名输入框
3. 坐标存入内存中的 session 对象
4. 显示当前已标记坐标数

**代码范围**:
- `public/shell.html` - 消息监听、弹窗 UI

**成功标准**:
- 点击页面任意位置 → 弹出命名框
- 输入名称 → 坐标保存到 session
- 状态栏显示已标记数量

---

### Step 3: YAML 生成与保存
**目标**: 将坐标导出为 YAML 文件

**任务**:
1. 设计坐标数据结构
2. 实现 YAML 序列化
3. 保存到 `~/.siliu/workspace/agents/`
4. 触发 hot-reload 加载 Agent

**代码范围**:
- `src/copilot/agents/dynamic-agent-loader.js` - 可能无需修改，复用现有 saveAgent

**成功标准**:
- 标记 3 个坐标 → 生成 YAML 文件
- 文件内容正确
- Agent 列表自动刷新

---

### Step 4: Agent 选择器下拉
**目标**: 左下角 Agent 选择器可展开下拉菜单

**任务**:
1. 重构 agent-footer 为下拉选择器
2. 实现 dropdown 显示/隐藏
3. 点击外部关闭 dropdown

**代码范围**:
- `public/shell.html` - CSS + JS

**成功标准**:
- 点击选择器展开下拉
- 显示所有 Agent 列表
- 点击其他区域关闭下拉

---

### Step 5: 创建 Agent 对话框
**目标**: 点击"创建新 Agent"打开对话框

**任务**:
1. 模态对话框 UI（名称、ID、图标、颜色）
2. ID 自动生成逻辑
3. 创建后自动进入标注模式

**代码范围**:
- `public/shell.html` - 模态框 UI

**成功标准**:
- 填写名称 → 自动生成 ID
- 选择图标/颜色
- 创建成功 → 自动开始标注

---

### Step 6: 跨页面标注
**目标**: 支持暂停→跳转→继续标注

**任务**:
1. 暂停按钮移除蒙版
2. 用户自由跳转页面
3. 继续按钮重新注入蒙版
4. 自动识别新页面类型

**代码范围**:
- `public/shell.html` - 状态管理

**成功标准**:
- 页面 A 标记 2 个坐标
- 暂停，跳转到页面 B
- 继续，标记 3 个坐标
- 保存后 YAML 包含两页数据

---
