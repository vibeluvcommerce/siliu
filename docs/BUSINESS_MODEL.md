## 四、法律与合规

### 4.1 用户协议要点

```
1. 禁止用途
   - 批量注册账号
   - 自动化刷量/刷单
   - 绕过网站安全措施
   - 抓取个人隐私数据

2. 频率限制
   - 单次任务最多 100 步
   - 操作间隔建议 > 2 秒
   - 24 小时内任务上限

3. 免责声明
   - 网站改版可能导致 Agent 失效
   - 过度使用可能导致账号限制
   - 用户需自行承担使用风险
```

### 4.2 合规策略

| 风险 | 缓解措施 |
|------|----------|
| **反爬虫法律风险** | 明确禁止高频操作；添加频率限制；用户协议免责声明 |
| **数据隐私** | 本地存储优先；不上传用户数据；匿名化日志 |
| **知识产权** | Agent 市场审核机制；侵权投诉处理流程 |
