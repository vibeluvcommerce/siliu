## 2. 核心自动化功能

### 2.1 浏览器控制操作

```javascript
// SiliuController 支持的操作
{
  navigate:   '页面导航',
  click:      '元素点击（selector/text）',
  clickAt:    '坐标点击（x, y）',
  type:       '文本输入',
  typeActive: '在活动元素输入',
  selectAll:  '全选文本',
  press:      '按键（Enter/Escape等）',
  scroll:     '页面滚动',
  wheel:      '滚轮事件（视频类网站）',
  hover:      '鼠标悬停',
  select:     '下拉框选择',
  upload:     '文件上传',
  screenshot: '截图',
  wait:       '等待',
  collect:    '数据采集',
  export:     '触发导出',
  done:       '任务完成'
}
```

### 2.2 上传模式

```
┌─────────────────────────────────────────────────────────┐
│                      上传流程                            │
├─────────────────────────────────────────────────────────┤
│  1. 用户发送上传指令                                     │
│           ↓                                             │
│  2. CDP 尝试直接设置 file input                         │
│           ↓                                             │
│  ┌─────────────────┐   失败    ┌──────────────────┐    │
│  │ CDP 直接上传    │ ────────→ │ 系统对话框拦截   │    │
│  │ （标准input）   │           │ （自定义上传）   │    │
│  └─────────────────┘           └──────────────────┘    │
│                                          ↓              │
│                              ┌─────────────────────┐   │
│                              │ 1. 设置待选文件      │   │
│                              │ 2. 点击上传按钮      │   │
│                              │ 3. 拦截系统对话框    │   │
│                              │ 4. 自动填充路径      │   │
│                              │ 5. 确认选择          │   │
│                              └─────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

### 2.3 数据采集与导出流程

```
用户: "采集这个表格数据"
           ↓
AI: 分析页面，找到表格
           ↓
AI: {"action": "collect", content: {...}, batchIndex: 0, hasMore: true}
           ↓
系统: 存储数据批次
           ↓
AI: 继续采集下一页...
           ↓
AI: {"action": "collect", ..., hasMore: false} 或超时
           ↓
系统: 自动触发导出
           ↓
文件保存到: ~/.siliu/workspace/exports/
```

## 6. 问题排查

### 6.1 坐标点击不准确

**症状**: AI 点击位置偏离目标

**排查**:
```javascript
// 检查 DPR
window.devicePixelRatio

// 检查视口尺寸
window.innerWidth, window.innerHeight

// 检查 BrowserView 尺寸
view.getBounds()
```

**常见原因**:
1. DPR 未正确计算
2. 视口尺寸不匹配
3. 页面缩放不是 100%

### 6.2 上传失败

**症状**: 文件无法上传

**排查步骤**:
1. 检查文件路径是否存在
2. 检查路径解析结果
3. 检查 CDP 是否连接
4. 检查拦截器是否运行

**日志检查**:
```
[SiliuController] upload: trying CDP mode...
[SiliuController] upload: CDP failed, trying system dialog interceptor...
[TabManager] File auto-selected: {...}
```

### 6.3 采集数据为空

**症状**: collect 后导出文件为空

**排查**:
1. 检查 collect action 的 content 格式
2. 检查 batchIndex 是否正确递增
3. 检查 hasMore 标志
4. 查看是否触发超时导出

### 6.4 AI 不执行操作

**症状**: AI 只回复文字，不控制浏览器

**排查**:
1. 检查 AI 服务连接状态
2. 检查是否在 Action 模式
3. 查看 AI 返回的 JSON 是否包含 action 字段
4. 检查 controller 是否初始化

### 6.5 导出文件损坏

**症状**: Excel/CSV 打不开或乱码

**排查**:
1. 检查数据编码（中文需 UTF-8）
2. 检查文件是否完整写入
3. 检查数组结构是否正确
