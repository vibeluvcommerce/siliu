# Siliu 自动化测试执行 - 完整流程

## 测试目标
在 Bilibili 完成一次完整的视频评论自动化流程

## 完整操作流程

打开 Siliu 浏览器，确保 AI 服务已连接（状态栏显示绿色已连接）。在 Copilot 侧边栏发送指令："打开B站视频 https://www.bilibili.com/video/BV1GJ411x7h7"，等待页面加载后视频开始播放。继续发送："向下滚动到评论区"，AI 会自动滚动页面直到评论区出现。然后发送："点击评论输入框"，看到输入框获得焦点后发送："输入'Siliu浏览器自动化测试评论'"，文字出现在输入框后发送："点击发送按钮"，等待评论发布成功后发送："截图确认评论是否显示在列表中"。整个流程应该无需人工干预，从打开视频到评论成功发布全部由 AI 自动完成。

## 预期结果
- 视频页面正常加载并播放
- 成功滚动到评论区
- 评论输入框正确获得焦点
- 文字正确输入
- 评论成功发布并显示在列表顶部
- AI 截图确认评论存在

## 如果卡住的常见原因
1. **页面加载失败** - 检查网络或 URL 是否正确
2. **找不到评论区** - 可能需要多滚动几次，或页面结构特殊
3. **输入框点不到** - B站评论区是动态加载的，可能需要等待
4. **发送按钮找不到** - 尝试指令改为"点击发表评论按钮"或"按回车发送"
5. **未登录** - B站评论需要登录，如果弹出登录框需要你先登录
