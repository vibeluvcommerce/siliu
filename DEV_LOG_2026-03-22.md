- **原因**: 更符合用户预期，避免误关闭整个浏览器窗口

## 技术细节

### 设置页面检测逻辑
```javascript
const existingSettingsTab = views.find(v => {
  if (!v.url) return false;
  // 检测 siliu://settings 协议
  if (v.url === 'siliu://settings') return true;
  // 检测 file://.../settings.html 路径（排除 copilot-settings.html）
  if (v.url.includes('settings.html') && !v.url.includes('copilot-settings.html')) return true;
  return false;
});
```

### 关闭当前视图 API
```javascript
function closeSettings() {
  if (window.siliuAPI?.closeCurrentView) {
    window.siliuAPI.closeCurrentView();
  } else {
    window.close();
  }
}
```
